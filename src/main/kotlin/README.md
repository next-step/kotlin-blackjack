### 설명
블랙잭 게임 구현
- 딜러와 플레이어 중 카드의 합이 21 또는 21과 가까운 숫자를 가지면 이김
- 카드의 숫자 계산은 카드의 숫자를 기본으로 함
  - Ace (1, 11), King, Queen, Jack (10)
- 게임 시작 시 두 장의 카드 지급
- 카드를 안받는다는 의사를 밝힐 때까지 계속 받을 수 있음 (단 합이 21이 넘으면 자동 패배)
