# kotlin-blackjack

## 2단계 - 블랙잭
### 요구사항
- [x] 13장의 카드가 있다
    - [x] Ace 는 1 또는 11이다
    - [x] King, Queen, Jack 은 10이다
- [x] 플레이어는 받은 카드 목록이 있다
    - [x] 카드는 추가 가능하다
    - [x] 카드목록의 합을 계산한다
        - [x] Ace 는 11로 계산한 합이 21을 초과하면 1로 계산한다

## 2단계 - 블랙잭(딜러)
### 요구사항
- [x] 딜러가 게임에 존재한다, 초기 카드를 같이 받고 결과도 같이 출력한다
- [x] 플레이어가 카드를 모두 받으면 딜러는 카드를 한장 더 받는다(16점 이하일 경우)
- [x] 딜러와 다른 플레이어들의 승패를 계산한다
