## 블랙잭 요구사항

1. 입력
   1. 플레이어 이름 (쉼표로 구분)
   2. 카드 뽑을지 말지 여부 (y,n으로 구분)
2. 카드 뽑기 조건
   1. 21점 미만인 경우 && 카드 뽑을지 말지 질문에 "y" 으로 대답한 경우
3. 게임 종료 조건
   1. 플레이어가 모두 카드를 뽑지 않는 경우 또는 모두 21점이 초과한 경우
4. 승리 조건
   1. 플레이어들 중 21점에 미만이면서 가장 가까운 플레이어가 승리
5. 카드별 점수
   1. 카드의 숫자 == 점수
   2. 예외 케이스 2개 존재
      1. 에이스는 1 또는 11로 계산
      2. King , Queen , Jack은 10으로 계산
