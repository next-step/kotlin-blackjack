# kotlin-blackjack

## 블랙잭

### 기능 요구사항

- [ ] 플레이어 이름을 입력받는다
    - [ ] 플레이어의 이름은 쉼표를 기준으로 구분한다
- [ ] 초기 카드를 지급한다
    - [ ] 덱은 일단 1개만 사용한다
    - [ ] 랜덤으로 두개의 카드를 지급한다
    - [ ] 지급하고 카드 현황을 출력한다
- [ ] 각 플레이어에게 카드를 추가로 지급할지 묻는다
    - [ ] y이면 카드를 추가로 지급한다
    - [ ] 플레이어가 n을 답할때까지 묻고 n을 답하면 카드 현황을 출력한다
    - [ ] 21을 넘을 경우 카드를 뽑지 못한다
- [ ] 게임 결과를 출력한다
