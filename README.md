# kotlin-blackjack

## step2 - 블랙잭
1. 게임 참가자의 이름을 입력받는다.
    - 1명 이상의 참가자가 필요하다.
2. 카드를 2장씩 분배한다.
3. 플레이어 별로 카드를 추가로 받을 것인지 묻는다.
    - y 또는 n 이외의 문자를 입력하면 재입력을 요구한다
5. 플레이어 별 카드 점수 합계를 계산한다.
    - Ace는 1 또는 11 중 더 큰 쪽으로 판정한다.
4. 가장 21에 가까운 숫자를 찾은 우승자를 찾는다.
    - 공통 우승이 존재할 수 있다.
    - 22 이상의 숫자를 만든 사람은 우승자가 될 수 없다
