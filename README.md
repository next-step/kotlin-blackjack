# kotlin-blackjack

# 기능 목록
- [ ] 게임에 참여할 사람의 이름을 입력받는 기능
- [ ] 입력받은 사람의 이름을 파싱하는 기능
- [ ] 참여할 사람이 2명 이하이면 예외를 던진다
- [ ] 카드를 랜덤으로 생성하는 기능
- [ ] 카드들의 합을 구하는 기능
- [ ] 보유한 카드를 출력하는 기능
- [ ] 카드를 더 받을 수 있는지 확인하는 기능
- [ ] 카드를 더 받는지 명령을 입력받는 기능
- [ ] 카드를 더 받는지 명령을 확인하는 기능
- [ ] 카드를 더 요청 하는 기능
- [ ] 게임 결과를 출력하는 기능
